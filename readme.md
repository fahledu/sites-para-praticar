# __Sites para praticar__

Esse repositorio visa centralizar sites gratuitos que vou encontrando onde possamos praticar nossas habilitades de desenvolvimento/programação.

Caso conheça algum que ache interessante incluir na lista, basta enviar umapull request ou abrir um issue.

Tentei categorizar o melhor que pude para facilitar a busca, mas alguns casos são relativos, ex JS que pode ter um exercicio exclusivo de JS ou fazer parte de um projeto de front-end por ex.... OU pode pedir algum framework, ou preprocessor CSS etc... Não fui tão detalista, então em JS pode ter React ou Jquery... em CSS pode ter SASS, etc... 

Caso esteja procurando uma linguagem que nao esteja listada, vale olhar algum dos sites de outras linguagens, por ex codewars que esta em c, java, possui uma lista de mais de 50 linguagens paa praticar...
